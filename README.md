### To-Do List
Aplicação web simples para gerenciar tarefas usando localStorage.
Link: https://oliveira-iago-temp.github.io/umc-lista-tarefas/

#
### Tecnologias
- HTML
- CSS
- JavaScript

#
### Funcionalidades
- Adicionar tarefa
- Remover tarefa
- Limpar todas as tarefas
- Salvar dados no localStorage
